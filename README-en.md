**JPatcher** is an IntelliJ IDEA plugin designed to quickly generate **incremental patch packages**.

The functionality of this plugin is extracted from [CPoot Tool](https://github.com/c-poet/cpoet-idea-tool).

Suggest pairing [PatchAssistant4J(补丁助手)](https://github.com/c-poet/patch-assistant-to-java) Use.

## Main Features 🌟

1. Select files to include in the patch from the editor window, project tree, VCS (Git/SVN) changes list, etc. 🗂️
2. Quickly generate an incremental patch package in ZIP format from selected files ⚡
3. Automatically generate a path description file 📝
4. Support for MapStruct generated files ♻️
