**JPatcher** 是一款专为IntelliJ IDEA开发的**增量补丁包**快速生成插件。

> 本插件功能提取自 [CPoet Tool](https://github.com/c-poet/cpoet-idea-tool)

## 主要功能 🌟

1. 支持从编辑窗口、项目树、VCS(GIT/SVN)变更等列表中选择需要生成补丁的文件 🗂️
2. 选择文件快速生成 ZIP 格式的增量补丁包 ⚡
3. 自动生成路径说明文件 📝
4. 支持MapStruct生成文件 ♻️
